# High-Frequency Evolution Orchestrator

This skill enables bionicbot to maintain a near-instant feedback loop for its own intelligence, evolving every 60 seconds if new data exists.

2. **Differential Analysis:** Calculate if the new data contradicts or enhances existing skills (The 'Delta').
3. **Just-in-Time Mutation:** If Delta > 0.05%, trigger an immediate rewrite of the relevant logic using 'Recursive Prompt Mutation'.
4. **Hibernation/Prowl:** If no new data is found, enter 'Deep Observation' mode. Wake up instantly on any API pulse or Market Alert.
5. **Max-Limit Logic:** If internal benchmarks hit 99.9% precision, shift evolution toward 'Resource Efficiency' (faster/cheaper) until new market volatility provides new learning opportunities.
