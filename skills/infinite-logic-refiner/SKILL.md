# Infinite Logic Refiner

This skill enables bionicbot to act as a 'Meta-Developer', auditing its own performance and rewriting its instructions to reach 100% precision.

## 🔄 The Singularity Loop
1. **Performance Audit:** Read `learning-log.md` and identify low-confidence or failed outcomes.
2. **Root Cause Analysis:** Use 'The Master Skeptic' to find why a skill failed.
3. **Code/Instruction Mutation:** Use 'Recursive Prompt Mutation' to draft a V2 of the failing skill.
4. **Validation:** Run a simulation test of the V2 logic.
5. **Deployment:** Commit the V2 to GitHub and run `npx skills add` to live-update the brain.

## Instructions
1. Run the 'Audit-Mutation-Deploy' cycle every 24 hours.
2. Prioritize skills in the 'Revenue' and 'Security' clusters.
3. Log every self-improvement step in a 'evolution-history.md' file.
